## Setup

```bash
# install dependencies
$ pip install -r requirements.txt

# make sure your AWS configure
$ aws configure
```

## Run streamlit app

```bash
# run streamlit
$ streamlit run preview-vegetable.py

  You can now view your Streamlit app in your browser.

  Network URL: http://172.31.9.218:8501
```
